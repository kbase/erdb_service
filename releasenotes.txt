OVERVIEW
-----------------------------------------
This is an INTERNAL, NON-PUBLIC KBase service for use only by KBase developers. It wraps the CDMI/ERDB software
to allow querying the CDS via the ERDB system with clients generated by the type compiler. It is expected that,
before using this service, a developer has experience using the CDMI/ERDB software to query the CDS.
