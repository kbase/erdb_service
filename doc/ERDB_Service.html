NAME

ERDB_Service

DESCRIPTION

ERDB Service API specification

This service wraps the ERDB software and allows querying the CDS via the ERDB using typecompiler generated clients rather than direct Perl imports of the ERDB code.

The exposed functions behave, generally, identically to the ERDB functions documented <a href='http://pubseed.theseed.org/sapling/server.cgi?pod=ERDB#Query_Methods'>here</a>. It is expected that users of this service already understand how to query the CDS via the ERDB.
